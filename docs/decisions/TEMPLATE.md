# NNNN. Short title in the imperative

- **Status**: proposed | accepted | superseded by [NNNN](NNNN-slug.md)
- **Date**: YYYY-MM-DD
- **Deciders**: names / roles

## Context

What is the situation, constraint, or problem this decision addresses?
Include what forced the decision now (not later), and any assumptions
that would change the answer if they were false.

## Decision

The choice made. Be specific — name the library, the pattern, the
concrete API surface.

## Alternatives considered

- **Alternative A** — why not (pros, cons, dealbreaker)
- **Alternative B** — why not
- ...

## Consequences

- **Positive**: what this buys us.
- **Negative**: what it costs. What becomes harder.
- **Follow-ups**: work this creates, tracked elsewhere.
